#  Le Bistro - Сайт ресторана

##  ОПИСАНИЕ ПРОЕКТА
Сайт ресторана французской кухни с возможностью бронирования столиков.
Сделан для мобильных телефонов.

Дата: Март 2026
Версия: 1.0

МОДУЛЬ 3: Форма бронирования
Где находится: index.html (строки 160-200), script.js (строки 70-140)

Что делает:

Принимает данные от клиента
Проверяет правильность ввода
Отправляет заявку

Поля формы:

Поле	       Что проверяет	                Если ошибка
Имя	         Не пустое, минимум 2 буквы	    "Введите имя"
Телефон	     10-11 цифр	                    "Введите телефон"
Дата	       Не раньше сегодня	            "Выберите дату"
Время	       С 12:00 до 23:00	              "Время с 12 до 23"

Код валидации:
'''javascript
// Из script.js
function validateName() {
  // Проверка имени
}
function validatePhone() {
  // Проверка телефона
}
